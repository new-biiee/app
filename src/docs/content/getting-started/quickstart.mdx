# Quickstart

Get from zero to your first settled trade in under 5 minutes.

## Prerequisites

- Node.js 18+
- Solana wallet (Phantom or Backpack)
- USDT (SPL) on the target network
- Carnot API endpoint (devnet or mainnet)

---

## Step 1 — Install the SDK

```bash
npm install @carnot/sdk
```

The SDK provides PDA derivation helpers, TypeScript types, the program IDL, and protocol constants.

---

## Step 2 — Connect Wallet and Deposit USDT

Open the Carnot app, connect your wallet, and navigate to **Account → Deposit**. This calls `trader_deposit` on `carnot_engine` and increases your on-chain vault balance in a `trader_account` PDA unique to your wallet. Withdraw anytime via **Account → Withdraw**.

---

## Step 3 — Fetch the Price Grid

```bash
GET /api/grid?market=BTC_USDT
```

Each cell in the response includes:
- `band`: `{ lower, upper }` in USDT
- `window_start` / `window_end`: Unix timestamps (5-second span)
- `reward_rate_bps`: payout multiplier in basis points (e.g., `15000` = 1.5×)
- `signature`: HMAC-SHA256 committing all fields before the window opened

---

## Step 4 — Place a Trade

```bash
POST /api/orders
{
  "market": "BTC_USDT",
  "cell": { ...cell data including signature... },
  "stake_usdt": 10,
  "direction": "IN"
}
```

You're predicting the settlement candle close falls **inside** the band when the 5-second window ends. The backend validates the cell signature, reserves your stake, and creates an order with status `PENDING`.

---

## Step 5 — Watch the Outcome

At `window_end`, the backend evaluates the 5-second OHLC candle's **close** against the band:

- Close **inside** $[\text{lower}, \text{upper}]$ → **WIN** — receive `stake × reward_rate_bps / 10000`
- Close **outside** → **LOSS** — stake goes to the vault

Poll `GET /api/orders/user` or watch the `orderResult` Socket.IO event.

---

## Step 6 — Claim Winnings After Batch Settlement

Outcomes are real-time; payouts are processed in **1-minute batches** (the keeper aggregates each minute and publishes on-chain). Once the keeper generates a ZK proof and the `batch_receipt` PDA is on-chain:

1. Fetch your Merkle proof from the backend
2. Submit `distribute_winnings` with the proof
3. USDT transfers from vault to your wallet

The app handles this via the **Claim Winnings** UI.
