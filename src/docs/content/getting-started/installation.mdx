# Installation

## Install the SDK

```bash
npm install @carnot/sdk
# or: yarn add @carnot/sdk  /  pnpm add @carnot/sdk
```

Peer dependencies:

```bash
npm install @solana/web3.js @coral-xyz/anchor
```

## Environment Variables

```bash
NETWORK=devnet                              # or mainnet
SOLANA_RPC_URL=https://api.devnet.solana.com
CARNOT_API_URL=https://api.devnet.carnot.fi
WALLET_KEYPAIR_PATH=~/.config/solana/id.json
```

---

## Basic Usage

```typescript
import { Connection, PublicKey } from '@solana/web3.js';
import { findCarnotTraderAccountPda, findCarnotVaultPda, CARNOT_PROGRAM_ID } from '@carnot/sdk';

const connection = new Connection(process.env.SOLANA_RPC_URL!, 'confirmed');
const programId = new PublicKey(CARNOT_PROGRAM_ID);
const walletPublicKey = new PublicKey('YOUR_WALLET_ADDRESS');

const [traderPda] = findCarnotTraderAccountPda(walletPublicKey, programId);
const [vaultPda] = findCarnotVaultPda(programId);

console.log('Trader PDA:', traderPda.toBase58());
console.log('Vault PDA:', vaultPda.toBase58());
```
