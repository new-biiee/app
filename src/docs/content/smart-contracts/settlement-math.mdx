# Settlement Math

These formulas are verified by ZK constraints (C4, D1, D3, D4) and re-enforced by the Solana program. Live parameters: `keeper_fee_bps = 25`, `protocol_fee_bps = 100`.

## Gross Payout (Per Winner)

$$\text{gross\_payout}_i = \frac{\text{stake}_i \times \text{multiplier\_bps}_i}{10{,}000}$$

`multiplier_bps = 10000` means 1× (return of stake only). `multiplier_bps = 15000` means 1.5×. Bounded to `[10000, market_max_multiplier]` by constraint A4.

## Net Payout (Batch Level)

$$\text{net\_payout} = \sum_{\text{winners}} \text{gross\_payout}_i - \sum_{\text{winners}} \text{stake}_i - \sum_{\text{losers}} \text{stake}_j$$

- `net_payout > 0` — pool pays out to winners on net
- `net_payout < 0` — pool gains from losers on net
- `net_payout = 0` — only fees move

## Keeper Fee

Charged on both directions:

$$\text{keeper\_fee} = \frac{|\text{net\_payout}| \times 25}{10{,}000}$$

Constraint D3 verifies this exactly.

## Protocol Fee

Only charged when pool gains (`net_payout < 0`):

$$\text{protocol\_fee} = \begin{cases} \dfrac{|\text{net\_payout}| \times 100}{10{,}000} & \text{if } \text{net\_payout} < 0 \\ 0 & \text{otherwise} \end{cases}$$

Constraint D4 enforces the conditional — a proof with non-zero `protocol_fee` when `net_payout ≥ 0` is invalid.

## Pool Balance Conservation

When `net_payout ≥ 0`:
$$\text{pool\_after} = \text{pool\_before} - \text{net\_payout} - \text{keeper\_fee}$$

When `net_payout < 0`:
$$\text{pool\_after} = \text{pool\_before} + |\text{net\_payout}| - \text{keeper\_fee} - \text{protocol\_fee}$$

Constraint D1 verifies this holistically. The program also checks `pool_after ≥ 0` and reverts if it would go negative.

## Example

3 winners (stakes $100, $200, $150; multipliers 1.5×, 2×, 1.2×), 2 losers (stakes $300, $250):

```
gross_payouts = 150 + 400 + 180 = 730
winner_stakes = 100 + 200 + 150 = 450
loser_stakes  = 300 + 250      = 550

net_payout = 730 - 450 - 550 = -270   ← pool gains

keeper_fee   = 270 × 0.0025 = $0.675
protocol_fee = 270 × 0.01   = $2.70

pool_after = pool_before + 270 - 0.675 - 2.70 = pool_before + $266.625
```
